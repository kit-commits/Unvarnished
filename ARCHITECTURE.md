# Architecture 

Unvarnished is currently an early stage MVP prototype focused on privacy-aware architecture, intentional feature scoping, and safe iterative development. 

## Current Stage

This repository documents the product convept, MVP scope, privacy principles, and initial UI screens. 

The current goal is to validate the core user experience before adding unnecessary technical complexity. 

## Design Principles

- Minimal attack surface
- Privacy-aware by default
- Clear user consent
- Limited data collection
- Simple, understandable user flows
- Iterative development before platform expansion
